## 1. 경쟁 구도 분석

### 1-1. 주요 경쟁사 DA 추정

| 도메인 | 추정 DA | 특징 |
|--------|---------|------|
| **creatrip.com** | 60~70 | 한국여행 특화, 동일 키워드 전방위 지배. "best dermatology Seoul", "head spa Seoul" 모두 1~3위 독점 |
| **reddit.com** | 90+ | UGC 특성상 롱테일에서 강세. 상업적 의도 키워드엔 약함 |
| **VisitSeoul.net** | 55~65 | 공식 관광청. 광고/예약 없어 클릭율 낮음 |
| **trazy.com** | 45~55 | 투어 패키지 중심, 뷰티 커버리지 낮음 |
| **seoulbeautytrip.com** | **추정 10~20** | 신규 도메인, 백링크 거의 없음 |

### 1-2. creatrip이 왜 이기는가

1. **콘텐츠 양**: 동일 카테고리에 50~100개 이상의 개별 업체 리뷰 페이지
2. **백링크**: 한국 여행 블로그, 유튜브, TikTok에서 수천 개 자연 링크
3. **UGC**: 사용자 리뷰 → 롱테일 키워드 자동 생성
4. **페이지 체류시간**: 갤러리, 리뷰, 예약 버튼 → 이탈률 낮음

### 1-3. seoulbeautytrip.com 현재 강점

1. **정부 허가 facilitator** — creatrip에 없는 신뢰 시그널
2. **WhatsApp 직접 예약** — 즉각적 전환 (UX 우위)
3. **업체별 상세 페이지 55개** — 구조화된 콘텐츠
4. **About 페이지 E-E-A-T 강화 완료** — Organization + FAQPage 스키마
5. **영어 전용** — 외국인 타겟, 경쟁자가 한/영 혼재
